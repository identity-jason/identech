Powershell - Export from FIM Portal to AVP

I’ve been meaning to write this for a while – it’s one of those scripts that I keep meaning to use and never quite spending 5 minutes to actually put together – very simply, some powershell that takes an XPath filter from the command line, queries the FIM portal to see what matches, and exporting the results into an AVP format.

Param ($filter)$uri = "http://localhost:5725/ResourceManagementService"$results = export-fimconfig -uri $uri -CustomConfig $filterforeach ($item in $results){            foreach ($attr in $item.ResourceManagementObject.ResourceManagementAttributes)    {        $line = $attr.AttributeName + ": " + $attr.Value        Write-Output $line    }    Write-Output ""    Write-Output ""}
