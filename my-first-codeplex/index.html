My first CodePlex...

I've been sat on a stack of tools for a while - certainly from before joining my current employer, and have been contemplating what to do with them - some I'll probably hang on to (and perhaps try turning into a 'product' of some sort) but others - well, I'd rather they were available for use rather than were just cluttering up my hard drives!

I've got the first one up on CodePlex now, which is my FIM (/ILM/MIIS) run profile sequencer "RunJob" - the concept is one I've seen elsewhere, but I don't think I've seen anything this comprehensive outside of a more coded solution.

Hopefully it'll prove useful for someone!

http://runjob.codeplex.com/
