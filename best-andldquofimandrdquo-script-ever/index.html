Best &ldquo;FIM&rdquo; Script Ever!

I’ve been using a lot of PowerShell with FIM and started thinking about which script or scripts actually did the most for me – and this week, it turns out it’s not PowerShell at all but a humble command file that restart’s the FIM Service and performs an issreset for me.

Not remotely glamorous but has probably saved me hours over the last few weeks!

net stop "forefront identity Manager service"net start "forefront identity Manager service"iisreset
